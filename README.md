## 🏪 Pobierz rozszerzenie

👉 Oficjalna wersja w Chrome Web Store:
https://chromewebstore.google.com/detail/instaling-bot/nhbenaecaphgicecneojaihcjbpglbdn

---

## 🚀 Instalacja (manualna)

1. Pobierz repozytorium
2. Wejdź w przeglądarce:

   ```
   chrome://extensions
   ```
3. Włącz **Tryb dewelopera**
4. Kliknij **„Załaduj rozpakowane”**
5. Wybierz folder z projektem

---

## ▶️ Jak używać

1. Wejdź na stronę:

   ```
   https://instaling.pl/app/
   ```

   lub:

   ```
   https://instaling.pl/ling2/
   ```

2. Kliknij ikonę rozszerzenia

3. Kliknij **Start Bot**

## ⚙️ Funkcje

### 🤖 Bot

* automatyczne wykonywanie zadań

---

### 💾 Zapisywanie danych

Słówka są automatycznie zapisywane w pamięci przeglądarki (`chrome.storage`).

* ✔ zapis w czasie rzeczywistym
* ✔ brak utraty danych
* ✔ współdzielone między popupem i botem

> ❌ Nie trzeba już ręcznie kopiować słówek

---

### 🧠 Zarządzanie słówkami

* **Show words** → wyświetla wszystkie słowa
* **Kliknij słowo** → edycja inline
* **X obok słowa** → usuń słowo
* **Hover na słowie** → pokazuje datę dodania

---

### ➕ Dodawanie słów

Dodaj własne tłumaczenia:

```json id="example1"
"kot":"cat","pies":"dog"
```

## 🛠️ Troubleshooting

### ❌ Słówka się nie zapisują

* sprawdź czy masz permission `"storage"`
* odśwież rozszerzenie:

  ```
  chrome://extensions
  ```

---

### ❌ Bot nie działa

* upewnij się, że jesteś na:

  ```
  https://instaling.pl/app/
  ```
* odśwież stronę

---

### ❌ Brak słów w popupie

* kliknij **Show words**
* wykonaj przynajmniej jedną sesję
